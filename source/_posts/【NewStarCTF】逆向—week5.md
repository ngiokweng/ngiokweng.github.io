## 拔丝溜肆 (easy)

拉入IDA，找到加密函數，發現是個魔改的base64，它在每次加密前先調用了`change_table()`，進入查看

![Untitled](Untitled.png)

根據隨機數來打亂base64的編碼表，而隨機數的亂數種已在`main`函數中給出，故可直接寫腳本解密

![Untitled](Untitled1.png)

這base64解密腳本是我之前學base64時手寫的，寫的很爛，但能用就行

```cpp
#include<iostream>
#include<string>
using namespace std;

string codeTableForBase32And64 = "ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789+/";

string table = "ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789+/";

/********************  BaseN解密   *************************/
string baseN_decode(string str, int N) {
	string s = table;

	//計算位數
	int bitSize = 0;
	while (N != 1) {
		N /= 2;
		bitSize++;
	}
	bitSize--;

	//逆向過程：
	string tmpStr = "";
	for (auto ch : str) {
		int i;
		if (ch == '=') {
			tmpStr += (char)0;
			continue;
		}

		for (i = 0; s[i] != ch; i++);
		tmpStr += (char)i;

	}
	int mark; //用來取得每個bit

	int tmp = 0;
	string res = "";
	int curBit = 7;

	//遍歷每個字符
	for (auto ch : tmpStr) {
		mark = 1;
		mark <<= bitSize; //只取後bitSize位

		while (mark) {
			//讀完8個bit後，將tmp值轉為字符串加到res中
			if (curBit < 0) {
				res += (char)tmp;
				tmp = 0;
				curBit = 7;
			}
			//2進制位轉為10進制
			if ((ch & mark) != 0) {
				tmp += pow(2, curBit);
			}
			curBit--;
			mark >>= 1;

		}
	}
	/* 處理最尾的部分 */
	if (curBit < 0) {
		res += (char)tmp;
	}
	return res;
}

void change_table_0()
{
	size_t result; // rax
	int v1; // [rsp+24h] [rbp+4h]
	int i; // [rsp+44h] [rbp+24h]

	v1 = rand() % 64;
	for (i = 0; i < 64; ++i)
	{
		table[i] = codeTableForBase32And64[(v1 + i) % 64];
	}
}

int main() {
	string enc = "CPaKBfUZFcNwW9qCKgyvuS2PGPQ9mttGc/wCNS0w6hDwGOSsOkOEkL5V";
	srand(0x1BF52u);
```

## E4sy_Mix (hard)

在`main`函數中看到下圖3處，可以看出在第3個紅框的地方進行了函數調用，而在此之前對`loc_402000`~`loc_402000+53`的地址進行了異或操作，所以可以看出這題考的是**SMC自解密**

![Untitled](Untitled2.png)

- 通常做法是用ida的腳本先將`loc_402000,loc_402000+53`這區域的值異或`0x54`，將其還原成`sub_402000`函數
- 點擊IDA左上的`File`→`Script command…`，Scripting language我用的是IDC

```cpp
auto addr = 0x402000;
auto i = 0;
for(i = 0;i<53;i++){
     auto val = Byte(addr+i);
     val = val^0x54;
     PatchByte((addr+i),val);
}
```

- 執行完腳本後，可以看到第1句匯編變成了`push ebp`，明顯符合函數的特徵
- 然後選中該範圍→按`u`
- 再選中該範圍→按`c`，選Analyze
- 最後再選中該範圍→按`p`生成為函數，然後就可以按`F5`了
