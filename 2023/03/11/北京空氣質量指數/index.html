【js逆向】某空氣質量網

發表於2023-03-11|更新於2023-03-11|Python
|閱讀量:

網址：http://www.aqistudy.cn/historydata/monthdata.php?city=北京

反爬手段

  1. F12檢測：用ctrl+shift+i，然後將開發者工具彈出
  2. 無限Debugger：用油猴注入以下hook腳本

分析過程

尋找關鍵點

  • 一開始在Network面板抓包時，沒有看到ajax請求( 但後來得知，只有在沒有網站緩存時會發送ajax請求來獲取數據 )
  • 之後查看Sources中的js代碼，也沒有找到想要的東西
  • 最後在網頁源代碼中，看到以下被加密的代碼

hook eval函數

  • 接下來就要想辦法查看eval中的內容，我使用的辦法是hook eval這個函數
  • 找到對應的那個eval函數，單步步入

分析eval函數

  • 單步步入eval函數後，就能看到”解密”後的內容，分析後發現suhI9pzg8akH6是我的目標，繼續跟進去

  • const dMqYT = gFEd9jX6phJLhqad(kQ8z, pw3c5ux)這句是從localStorageUtil中獲取數據，相當於從網站緩存中獲取，因此只有第1次訪問該網站或刪除網站緩存後，才會發送ajax請求來獲取數據

  • 先刪除網站緩存：開發者工具→應該程式→儲存空間→清除
  • 然後刷新頁面，重新去到發送ajax的地方

加密函數分析

  • 之後對加密函數進行分析，先看加密參數是怎樣來的

  • 經多次測試後發現，下圖兩處地方是會改變的
  • 第1個地方：每次隨機生成不同的字符串
  • 第2個地方分3種情況：
    1. 只有BASE64
    2. BASE64+AES
    3. BASE64+DES
  • 因此寫腳本要注意分3種情況，並且每次AES、DES加密所用到key和iv都會變，所以要動態獲取

解密函數分析

  • 請求成功後，用dobXiBdA52b函數對返回的結果進行解密，解密後的數據就是我想要的空氣質量數據

  • 可以看到該函數就是簡單的BASE64、DES、AES解密，只需找到key和iv就能輕易地解密
  • 且經過多次測試發現該函數的形式不會改變，只有key和iv會變
  • 因此也需要動態地獲取key和iv

腳本編寫

思路

  1. 因為所有的key和iv都會變，因此需要動態獲取js代碼
  2. 而js被加密，所以要對js代碼進行解密
  3. 獲取AES、DES的key和iv( 在解密後的js代碼最上方 )
  4. 為加密函數、解密函數匹配對應的key、iv ( 加密、解密所用到的key、iv不同，要進行匹配 )
  5. 模擬加密函數進行加密
  6. 模擬解密函數進行解密

完整代碼

  • 說明：main.py是主要的python代碼，main.js和func.js是用來模擬加密、解密過程的代碼，myHttp.js提供了幾個api接口，讓main.py能執行main.js中的js代碼

main.js

myHttp.js
